Fraud Detection in Ethereum Cryptocurrency Transactions

This project focuses on detecting fraudulent transactions in Ethereum cryptocurrency using machine learning. An XGBoost classifier was trained on blockchain transaction data, achieving high accuracy and a strong ROC-AUC score. Stratified sampling and precision-recall analysis were used to handle class imbalance and improve detection of rare fraud cases. The model enhances the reliability of blockchain analytics by identifying suspicious activities effectively.

Customer Churn Prediction using Machine Learning

The Customer Churn Prediction project uses machine learning to identify customers likely to leave a telecom service. Using the Telco Customer dataset, the pipeline includes data cleaning, feature engineering, and model training. Techniques like SMOTE are applied to handle class imbalance, and models are optimized through hyperparameter tuning. Evaluation is done using the ROC-AUC score to ensure high predictive accuracy. Additionally, churn trends are visualized using Seaborn to uncover valuable business insights.
